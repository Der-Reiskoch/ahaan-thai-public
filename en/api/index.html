Developer Tools

Application Programming Interfaces

DE/EN

Thai Food Encyclopedia API

This interface provides the complete Thai Food Encyclopedia in JSON format. It contains structured data on hundreds of Thai dishes with detailed information such as a description of the dish, German and English translations, Thai names, regional categorisations and links to recipes.

Thai Food Dictionary API

This API provides the Ahaan Thai Food Dictionary in JSON format. It contains culiary Thai terms and divides these into different categories (e.g. spices, vegetables, meat, preparation methods). Each entry contains the word in Thai, German and English translations as well as romanisation in German and English spelling. The dictionary makes it possible to systematically look up Thai kitchen terms and translate between Thai, German and English.

Thai Cook Book Info API

This API interface provides the meta information for my Thai Cookbooks in JSON format. Each entry contains information such as author, description, publisher, ISBN, year of publication, level of difficulty as well as image and shopping links. The collection includes German as well as English and Thai cookbooks.

Thai Cook Book Library API

This interface provides my collection Thai Cookbooks and the recipes it contains in JSON format. For , the library contains information for each recipe such as the corresponding book, page numbers, assignment to a region, Thai script, German and English titles and romanisations. It also contains image paths and URLs for the individual recipes, providing a comprehensive digital recipe database with multilingual support.

To make these interfaces available for AI tools, there is the project Ahaan Thai MCP Server on github.com. You can download it to integrate the included MCP servers locally into your AI tool (e.g. Claude).
