Chu Chee Plaa

Fisch in dicker Currysoße

ฉู่ฉี่ปลา Südthailand DE/EN
Curry Fisch

Bei “Chu Chee Plaa” ( ฉู่ฉี่ปลา ) handelt es sich um gebratenen oder gegrillten Fisch, der mit einer dickflüssigen Soße aus Kokosmilch auf Basis einer roten Currypaste serviert wird. Der Name „Chu Chee“ kommt angeblich von dem Geräusch, das die Currypaste macht, wenn sie in der heißen Kokosmilch brutzelt.
