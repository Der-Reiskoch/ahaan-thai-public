Luk Chin

Fleisch- oder Fischbällchen

ลูกชิ้นZentralthailand
Rindfleisch Schweine­fleisch Fisch Meeres­früchte Garnelen

Als “Luk Chin” ( ลูกชิ้น ) werden in Thailand Bällchen bezeichnet, die aus unterschiedlichen Sorten Fleisch wie Schwein, Rind, Fisch oder Garnelen zubereitet werden. Das Fleisch wird fein gehackt, mit Mehl vermischt, gewürzt und dann zu kleinen Kugeln geformt, die anschließend gekocht oder frittiert werden. Die Luk Chin werden meist in Suppen, Currys oder als auch Snack mit einem Dip serviert.

Spezifische Bezeichnungen

  • Schweinefleisch: “Luk Chin Mhu” ( ลูกชิ้นหมู )
  • Rindfleisch : “Luk Chin Nuea” ( ลูกชิ้นเนื้อ )
  • Fisch : “Luk Chin Plaa” ( ลูกชิ้นปลา )
Alternative Bezeichnungen
Luk Jin, Luk Shin, Look Chin
Verwendung
  • Luk Chin Ping
  • Gaeng Djued Luk Chin
