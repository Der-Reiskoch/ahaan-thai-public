Gaeng Phed Nuea

Rotes Curry mit Rindfleisch

แกงเผ็ดเนื้อZentralthailand
Rindfleisch Curry

Als “Gaeng Phed Nuea” ( แกงเผ็ดเนื้อ ) wird ein Curry mit Kokosmilch auf Basis einer roten Currypaste bezeichnet, das mit Rindfleisch zubereitet wird. Meist wird neben dem Fleisch auch noch diverses Gemüse als Einlage verwendet. Solche Currys werden auch als “Gaeng Daeng Nuea” ( แกงแดงเนื้อ ) bezeichnet.
