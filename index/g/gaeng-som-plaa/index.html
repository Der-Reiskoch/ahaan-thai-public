Gaeng Som Plaa

Säuerlich-scharfe Suppe mit Fisch

แกงส้มปลาSüdthailand
Suppe Fisch

“Gaeng Som Plaa” ( แกงส้มปลา ) ist eine südthailändische saure Suppe mit Fisch. Die Suppe basiert auf einer Paste aus Tamarinde, Chilis und anderen Zutaten. Oft werden in einer Gaeng Som noch diverse Gemüse mit gekocht, wie zum Beispiel Lotusstängel oder Tarostängel, welche in der Lage sind, viel Flüssigkeit aufzunehmen.
