Gai Hor Bai Toey

Hähnchen im Pandanblatt

ไก่ห่อใบเตย Zentralthailand
Geflügel Pandan Gegrillt Frittiert

Bei “Gai Hor Bai Toey” ( ไก่ห่อใบเตย ) handelt es sich um mariniertes Hähnchenfleisch, das in Pandanblätter gewickelt wird. Diese Pandan-Päckchen werden dann gegrillt oder frittiert, wobei das Fleisch nicht nur die Aromen der Marinade, sondern auch der Blätter annimmt. Das Gericht ist auch unter dem verkürzten Namen “Gai Bai Toey” ( ไก่ใบเตย ) zu finden.

Das Gericht wurde erstmals in den 1980er Jahren in einem damals sehr bekannten Restaurant, dem “Djit Pochana” ( จิตรโภชนา ) in Bangkok, serviert. Es war so beliebt, dass unzählige Restaurants in der Stadt es kopierten und “Gai Hor Bai Toey” so schließlich in thailändischen Restaurants auf der ganzen Welt Verbreitung fand.
