Pad Pak Kanaa Mhu Grob

Chinesischer Brokkoli mit Schweinebauch

ผัดผักคะน้าหมูกรอบ Zentralthailand
Wokgericht Schweine­fleisch Schweinebauch Brokkoli

“Pad Pak Kanaa Mhu Grob” ( ผัดผักคะน้าหมูกรอบ ) ist ein Wokgericht bei dem knuspriger Schweinebauch zusammen mit chinesischem Brokkoli angebraten wird. Oft wird der Name des Gerichts auch auf “Pad Kanaa Mhu Grob” ( ผัดคะน้าหมูกรอบ ) oder gar “Kanaa Mhu Grob” ( คะน้าหมูกรอบ ) verkürzt.
