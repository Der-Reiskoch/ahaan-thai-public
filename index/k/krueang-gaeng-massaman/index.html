Krueang Gaeng Massaman

Currypaste für Massaman Curry

เครื่องแกงมัสมั่น Südthailand DE/EN
Currypaste

Die Currypaste mit dem Namen “Krueang Gaeng Massaman” ( เครื่องแกงมัสมั่น ) dient als Basis für südthailändische Massaman Currys. Sie besteht aus Zutaten wie getrockneten Chilischoten, Zimt, Kreuzkümmel, Koriandersamen, Kardamom und Muskat, die im Mörser zerstoßen werden. Diese Paste wird in Thailand auch “Prik Gaeng Massaman” ( พริกแกงมัสมั่น oder “Nam Prik Gaeng Massaman” ( น้ำพริกแกงมัสมั่น ) genannt.
