Krueang Gaeng Gari

Gelbe Currypaste

เครื่องแกงกะหรี่ Zentralthailand DE/EN
Currypaste

Als “Krueang Gaeng Gari” ( เครื่องแกงกะหรี่ ) wird in Thailand gelbe Currypaste bezeichnet. Diese wird aus getrockneten Chilischoten, Kurkuma, Koriandersamen und Kreuzkümmel sowie weiteren Gewürzen hergestellt. Die Paste wird auch “Prik Gaeng Gari” ( พริกแกงกะหรี่ ) oder “Nam Prik Gaeng Gari” ( น้ำพริกแกงกะหรี่ ) genannt.

Die Intensität sowie der Ton der gelben Farbe der Paste hängen von den verwendeten Zutaten ab. So können Currys, die mit einer gelben Paste zubereitet werden, farblich zwischen einem rötlichen Ton bis zu einem sehr knalligen Gelb variieren.
