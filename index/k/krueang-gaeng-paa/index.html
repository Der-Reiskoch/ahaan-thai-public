Krueang Gaeng Paa

Currypaste für Jungle Curry

เครื่องแกงป่าNordthailand, Nordostthailand
Currypaste

“Krueang Gaeng Paa” ( เครื่องแกงป่า ) ist eine Variante der roten Currypaste, bei der oft neben den typischen getrockneten Chilis auch frische grüne Chilis verwendet werden. Daneben kommen Zutaten wie Zitronengras, Galgant und Krachai zum Einsatz.

Diese Gewürze werden je nach Rezept vor dem Zerstoßen noch geröstet, was der ganzen Paste ein mehr oder weniger starkes Röstaroma verleiht. Die Paste wird auch “Prik Gaeng Paa” ( พริกแกงป่า ) oder “Nam Prik Gaeng Paa” ( น้ำพริกแกงป่า ) genannt.

Alternative Bezeichnungen
Krueang Gaeng Pa, Krueang Gaeng Bpa, Krueang Kaeng Paa, Prik Gaeng Paa, Nam Prik Gaeng Paa
Eine Variation von
Krueang Gaeng Phed
Verwendung
  • Gaeng Paa Nuea
  • Gaeng Paa Mhu Paa
