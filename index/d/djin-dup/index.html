Djin Dup

Gehämmertes Fleisch

จิ้นตุ๊บ Nordthailand DE/EN
Rindfleisch Schweine­fleisch

Beim “Djin Dup” ( จิ้นตุ๊บ ) handelt es sich um gegrilltes Rindfleisch oder Schweinefleisch, das mit einem Hammer in fasrige Stücke geklopft wird. Oft wird das Fleisch vor dem Grillen mariniert und dann sehr lange auf dem Grill belassen. Handelt es sich um Schweinefleisch, so wird das gehämmerte Fleisch auch als “Mhu Dup” ( หมูตุ๊บ ) bezeichnet. Diese Zubereitungsmethode aus Nordthailand ermöglichte früher den Verzehr von Fleisch, das ansonsten zu zäh gewesen wäre.
