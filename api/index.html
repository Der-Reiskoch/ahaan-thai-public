Application Programming Interfaces

DE/EN

Thai Food Encyclopedia API

Diese Schnittstelle stellt das komplette Thai Food Lexikon von ahaan-thai.de im JSON-Format bereit. Sie enthält strukturierte Daten zu hunderten von thailändischen Gerichten mit detaillierten Informationen wie einer Bechreibung des Gerichts, deutschen und englischen Übersetzungen, Thai-Namen, regionalen Zuordnungen und Links zu Rezepten.

Thai Food Dictionary API

Diese API stellt das Ahaan Thai Wörterbuch im JSON-Format bereit. Dieses enthält Thai-Begriffe aus der Küche und unterteilt diese in verschiedene Kategorien (wie z.B. Gewürze, Gemüse, Fleisch, Zubereitungsarten). Jeder Eintrag enthält das Wort auf Thai , deutsche und englische Übersetzungen sowie Romanisierung in deutscher und englischer Schreibweise. Das Wörterbuch ermöglicht es, thailändische Küchenbegriffe systematisch nachzuschlagen und zwischen Thai, Deutsch und Englisch zu übersetzen

Thai Cook Book Info API

Diese API-Schnittstelle liefert die Metainformationen zu meinen thailändischen Kochbüchern im JSON-Format. Jeder Eintrag enthält Informationen wie Autor, Beschreibung, Verlag, ISBN, Erscheinungsjahr, Schwierigkeitsgrad sowie Bild- und Shopping-Links. Die Sammlung umfasst sowohl deutsche als auch englische und thailändische Kochbücher.

Thai Cook Book Library API

Diese Schnittstelle stellt meine Sammlung thailändischer Kochbücher und die darin enthaltene Rezepte im JSON-Format bereit. Die Bibliothek enthält für jedes Rezept Informationen wie das zugehörige Buch, Seitenzahlen, Zuordnung zu einer Region, thailändische Schrift, deutsche und englische Titel sowie Romanisierungen. Zusätzlich sind Bildpfade und URLs zu den einzelnen Rezepten enthalten, wodurch sie umfassende digitale Rezeptdatenbank mit mehrsprachiger Unterstützung zur Verfügung stellt.

Um diese Schnittstellen für KI Tools verfügbar zu machen, gibt es das Projekt Ahaan Thai MCP Server auf github.com. Du kannst es herunterladen um die enthaltenen MCP Server lokal in dein KI Tool (z.B. Claude) einzbinden.
